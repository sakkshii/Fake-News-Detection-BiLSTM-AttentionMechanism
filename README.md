# Fake News Detection — BiLSTM & Attention-BiLSTM

Replication and extension of:
Padalko, Chomko & Chumachenko (2024)
"A novel approach to fake news classification
using LSTM-based deep learning models"
Frontiers in Big Data

## Models
- Model 1: Bidirectional LSTM (BiLSTM)
- Model 2: Attention-Based BiLSTM
  - Custom attention layer from scratch

## Dataset
- ISOT Fake News Dataset
- 44,898 articles (Fake.csv + True.csv)

## Tech Stack
Python | TensorFlow/Keras | Google Colab
